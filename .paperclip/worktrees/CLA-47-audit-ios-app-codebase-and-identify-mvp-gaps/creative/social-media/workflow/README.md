# Social Media Scheduling Workflow

## Ferramenta Recomendada: Later ou Buffer

Ambas oferecem:
- Agendamento multi-plataforma
- Calendário visual
- Approvals workflow
- Versão gratuita para startups

## Workflow de Publicação

### 1. Criação de Conteúdo (Semana -1)
```
CMO → Brief de conteúdo
       ↓
Criativo → Rascunho/Design
       ↓
Revisão → Ajuste se necessário
```

### 2. Aprovação (Semana -1)
```
Conteúdo pronto → Post no slack #marketing-approval
                      ↓
Aprovação (CMO/Núcleo) → Aprovar ou Solicitar Mudança
```

### 3. Agendamento (Semana atual)
```
Aprovado → Agendar no Later/Buffer
               ↓
Verificar Timing → Postar conforme calendário
```

### 4. Monitoramento (Pós-publicação)
```
Post Ao Vivo → Monitorar engajamento (24h)
                   ↓
Report → Adicionar ao weekly report
```
